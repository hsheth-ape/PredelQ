# Session Logs

This directory contains session logs from Planning, Execution, and UX sessions.

## Naming Convention

```
{DATE}-{type}-{NNN}.md

Examples:
- 2025-01-01-planning-001.md
- 2025-01-01-execution-sprint3.md
- 2025-01-02-ux-001.md
```

## Lifecycle

1. **Create** — Start of session
2. **Update** — Add milestones during session
3. **Complete** — Mark final status
4. **Review** — Planning reviews for promotions
5. **Promote** — Move valuable items to permanent docs
6. **Archive** — Old logs can be deleted after promotion
